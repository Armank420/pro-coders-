Steps to upload a file.

1. Select the File:
    Click the upload button.
    A file dialog box will appear.
    Browse your computer or device to find the file you     want to upload.

2. Choose the File: Click on the file to select it, then      click “Open” or “Choose” in the dialog box.

3. Confirm the Upload: Depending on the platform, you      may need to confirm or start the upload. Some       services will show a progress bar.

4. Hence, you have uploaded the content.
